The Count wishes to play a game.

You will be given a random number to count to at the beginning of the game.

Each of the four pictures of the Count has a hidden number value assigned to it.

Clicking on them will add that value to your running total.

You win by matching your running total to the number the Count demands.

You lose if you exceed the number the Count demands.

Each time the game restarts, the values for the pictures will change.

He's a vampire who's clearly lost his mind. Hope you don't lose and upset him...
